# linux95apps
Small and fast components that mimic windows 95 on linux,
Using fltk (fast light tool kit),
these programs fit& run with [Chicago95](https://github.com/grassmunk/Chicago95) theme environment
such as xfce, icewm, etc,

---
Gemini 3.1 used with low level intraction and human (me) code reviews.

<img src="./.github/f3be892b-3ca9-4940-8722-876fb93cbcaf" width="256">

## fltk-devel :

|arch|debian|fedora|
|----|----|----|
|`pacman -S fltk`|`apt install libfltk1.4-dev`|`dnf install fltk-devel`|

build >> `make`
